R Projects and Skills

Portfolio Overview

These assignments show a progression from foundational R programming to end-to-end statistical analysis. Taken together, they highlight how I use R to import and clean data, reshape messy files into tidy datasets, produce static and interactive visualizations, run simulations, fit regression models, and communicate results through reproducible HTML reports.

Core R Skills Demonstrated

  • Data import and cleaning: readr, readxl, janitor, missing-value handling, recoding, and joins
  • Data transformation: dplyr, tidyr, separate(), pivot_longer(), pivot_wider(), grouped summaries
  • Visualization: ggplot2, patchwork, plotly, flexdashboard, figure saving for reports
  • Statistical inference: t.test(), prop.test(), confidence intervals, and interpretation of uncertainty
  • Simulation and resampling: custom functions, replicate(), Monte Carlo simulation, bootstrap
  • Modeling and validation: lm(), glm(), broom, modelr, cross-validation, and RMSE comparison
  • Communication: R Markdown workflows, project organization, and portfolio-style reporting
